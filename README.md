## Installation and Usage

### Frontend

The frontend is a SvelteKit app, which requires Node to run.

```zsh
cd ./frontend/garticband
npm i
npm run dev
```

### Backend

The backend is a JavaScript server developed in the Node runtime.

```zsh
cd backend
npm i
npm run dev
```
